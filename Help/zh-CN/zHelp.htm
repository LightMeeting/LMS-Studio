1、关于来米工坊（LMS）

马太福音18:20 “因为无论在哪里，有两三个人奉我的名聚会，那里就有我在他们中间。”

本项目是为基督信众，进行网上读经与团契，而开发的互联网边缘通讯软件。本项目核心思想是去中心化类似Zoom的会议软件，由超轻量主持端电脑与少数多线程客户端组成。

读经：英语（新国际版）与中文（和合本）圣经预装在此软件。将来有望支持其他语言版本。

团契：基于网络速度，支持至多24人参会，可以支持10-20人的团契聚会。通讯目前限于音频、视频与文字聊天。

最新软件版本为英语版（2.1.0）以及中文版（2.1.1）。

1) LMS如何工作

LMS有一个主持人及多个参会人。

首先，主持人基于其本地服务设置生成一段圣经章节，作为链接用以聚会。圣经章节例如：“历代志上2:41-19:19 约翰福音8:10-12:50”。

其次，主持人启动会议链接后，LMS系统将自动生成一个会议口令。主持人请使用其他通讯工具如电邮或短信，分别发送圣经章节及口令给参会人。

再次，参会人输入含圣经章节及口令的链接，加入会议。

最后，将启动系统默认的浏览器连接到会议。

2) 通讯结构

LMS使用树状通讯结构，主持人位于树顶，其他参会人位于树枝，有上枝下枝。一般情况下，唯有主持人讲话，其余参会人通过举手要求讲话。当主持人讲话时，参会人也可以同时讲话，但主持人可否听到您的发言基于您的座位远近以及他的会议设置。

 

2、何处下载与如何安装LMS

LMS目前支持Windows 10的64位与32位电脑。将来计划支持安卓手机。

许可政策是 双许可，GNU通用公共许可证或着商用许可。

在门户lightmeeting.github.io下载LMS.

可执行文件的默认安装文件路径为 “C:\Program Files\LM Fellowship\LMS Studio\Release\LMS_Studio.exe”，

其中，“Release”指代版本号，如1.0.0作为第一版。

假如您卸载LMS，本程序将被移除，但是您的设置数据将保留在默认目录“C:\Users\Username\AppData\Roaming\LightMeeting”，

其中，“Username”指代Windows登录名。您可以手动移除该数据。

 

3、首次运行LMS

请允许软件通过Windows防火墙。

 

4、开始聚会

1）在文件菜单下启动开始聚会命令，或者可以从视窗左上方启动。

2）选择正确的圣经书目章节，然后点击确定键。

3）登录对话框有两种登录模式：主持人或者参会人

 选择正确的登录模式，键入您中意的用户名，点击应用键。

4）主持人模式

请拷贝圣经章节（历代志上2:41-19:19 约翰福音8:10-12:50）与口令，分别发送给其他参会人员。

5）参会人模式

请键入收到的口令。

 

5、语言设置

可以在文件菜单下找到设置。

目前全面支持仅限于英语与中文。

 

6、本地服务设置

1）本地服务设置是为主持会议使用

在文件菜单下找到本地服务设置。如果您只是参会人员，这是个可选项，您可以忽略。您可以跳过整个第6章。

输入本地IPv4的公开地址，根据格式XXX.XXX.XXX.XXX（例如185.199.111.153）。下文将告诉如何找到您的本地IPv4的公开地址。

输入主要端口号，例如5150。

次要端口号将从主要端口号自动生成，公式为 主要端口号 + 4。

这两个端口是下文稍后您将允许的端口转发。

 

2）万维网服务

若是您主持会议，您需要打开本地万维网服务。

默认万维网端口号是8080，您可以选择一个不同于主要端口号以及次要端口号的数字。

 

3）如何找到本地IPv4的公开地址

A）点击“获取IP地址”键；

B）或者使用您的浏览器，点击本地网址

ipify.org javascript api

或者网站，例如

https://whatismyipaddress.com/

https://www.ip138.com/

 

4）如何允许调制解调器做端口转发

使用您的浏览器，登录到调制解调器（或者叫猫）的主页，网址类似如下

http://192.168.0.1

http://192.168.1.1

登录后，如下图进行端口转发，将默认主要端口号5150以及次要端口号5154的TCP/UDP流量转发到您的电脑（192.168.0.15）。或者也可以使用上文您中意的其他端口号。

 

7、测速设置

测速设置是为建立更加可靠的网络。

 

8、使用说明

1）会议厅

会议厅可供会议中使用。在会议厅里，可以看到主讲以外的其他参会人员。

在会议厅，主持人还负责协调会众发言。主持人点击选择某参会人对会众发言。

在该参会人结束发言时，主持人可以点击他自已的参会方框，允许他重新发言。

另外，可以点击主屏方框上的箭头全屏播放，若是您想更好地欣赏主讲的讲话或是阅读他分享的远程屏幕。

 

2）聊天室

聊天室可以聊天，可以是与所有人或是个别人。

 

3）设置台

您可以设置相机、可以捕获网页、视窗甚至整个屏幕，也可以看到统计数据。

 

4）读经阁

读经阁是供您在大屏幕阅读圣经，无论是否开会。相比常用的读经软件，本读经阁的主要优势是一书一屏，让阅读圣经有整体感，方便关键词查询。

 

9、聚会时有效通讯的诀窍

1）网络速度

节省带宽是有效通讯的关键。当主持人在讲话时，参会人最好关闭他们各自的音频/视频。

默认音频速度是160 bit/s，默认视频速度是400 bit/s，总之为每个参会人员，LMS需要至少 70K Byte/s。至多7方需要单向500K Byte/s，以及双向1M Byte/s。

2）通讯次序

当前软件鼓励开会时，一个时间只有一位在主讲。

一般情况下，只有主持人对其他参与会众发言。

在会议厅，主持人还负责协调会众发言。主持人点击选择某参会人对会众发言。

在该参会人结束发言时，主持人可以点击他自已的参会方框，允许他重新发言。

3) 如何修整我的视频

如果LMS在首次运行时没有发现您的摄像头，而您确认有摄像头。请重启电脑，启动LMS，您现在将可能发现有摄像头。若还是没有摄像头，您可以试着清除浏览器的浏览数据。

4）如何修整主讲的视频

如果当前主讲的视频模糊，可以点击“聚焦讲员”按钮。

5）如何修整回声

您可以点击任何方框的扬声器按钮静音，而关闭回声。

 

10、现存的问题

1）音频-视频延时

一般情况下，不到1秒延时。基于网络速度，可能有更多的延时。

2）多方通讯不可靠

在某些网络状况下，多方通讯时，参会人员有可能听不清楚主持人讲话。

3）连线问题

有时，参会人输入错误的圣经章节及口令，导致无法登录。

有时，参会人在看到成功登录窗口后，还是无法加入会议，可能是带宽等因素导致。请等待一两分钟，重启LMS再试。好运！
